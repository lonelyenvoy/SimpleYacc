# SimpleYacc
A simple grammar parser generator for LL(1) Grammar

简单LL1语法分析器生成器

### 使用方法

运行根目录下的run.bat，或使用命令行

```batch
cd SimpleYacc
./run.bat
```
即可看到测试结果。

### 项目依赖于
[JDK 1.8+](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
